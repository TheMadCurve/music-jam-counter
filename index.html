How to Add to OBS

  1. Open OBS Studio
  2. Click the + button in the Sources panel
  3. Select Browser
  4. Paste the URL above into the URL field
  5. Set width to 400 and height to 200
  6. Check Shutdown source when not visible
  7. Click OK
